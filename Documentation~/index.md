# NanoRes Solana Toolbelt

Welcome to the NanoRes Solana Toolbelt documentation hub. This package bundles runtime and editor tooling for connecting Unity projects to the Solana blockchain and NanoRes' Owner Governed Asset Ledger (OGAL).

## Contents

- `solana-toolbelt-overview.md` – high-level description of the toolbelt and its components.
- `solana-unity-sdk-evaluation.md` – evaluation notes for the upstream Solana Unity SDK.
- `qa/` – quality assurance reports and test plans.

## Getting Started

1. Install the package through the Unity Package Manager from Git or the Asset Store.
2. Review the **Solana Wallet** sample (`Samples~/Solana Wallet`) and import it via the Package Manager **Samples** section.
3. Use the prefabs in `Runtime/Resources/SolanaUnitySDK` to embed wallet flows in your scenes.
4. Configure OGAL services using the assets located under `Runtime/Toolbelt/_Data`.

For more details explore the documents listed above or contact NanoRes Studios via the repository issue tracker.
